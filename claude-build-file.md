# BIG GREEN T'S — CLAUDE CODE BUILD FILE

This file is designed to be used directly with Claude Code to generate a production-ready website.

---

## PROJECT OVERVIEW

Build a clean, modern website for:

Big Green T's  
Custom Screen Printing & Embroidery Since 1987

This is a family-owned, New Hampshire-based business focused on:
- Screen printing (primary)
- Embroidery (secondary)
- On-site event merchandising (key differentiator)

---

## DESIGN DIRECTION

The website MUST feel:

- Established
- Local
- Trustworthy
- Straightforward
- Not corporate
- Not flashy
- Not startup/SaaS-like

Visual rules:
- Use Dartmouth green (#00693e or similar) as primary color
- White background
- Clean typography (Inter, Helvetica, or system font)
- Generous spacing
- Minimal UI elements
- No gradients, no glassmorphism, no gimmicks

---

## TECH STACK

- Next.js (App Router)
- Tailwind CSS
- Responsive (mobile-first)
- No external UI libraries unless necessary
